会議議事録システム

AIを活用した自動文字起こし・議事録生成システムです。

機能

  • リアルタイム音声認識による文字起こし
  • AIによる議事録の自動要約
  • 会議のステータス管理
  • 議事録の詳細表示

使い方

  1. 「新しい会議を作成」ボタンをクリック
  2. 会議タイトルを入力
  3. 録音を開始して会議を開始
  4. 録音終了後、AIが自動で議事録を生成

技術仕様

  • バックエンド: Django
  • フロントエンド: HTML/CSS/JavaScript
  • データベース: SQLite
  • AI: Web Speech API + 独自処理
