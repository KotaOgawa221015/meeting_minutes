📋 Meeting Minutes & Debate System

AIを活用した会議議事録生成とディベートシステム

📱 システム概要

このアプリケーションは、2つの主要機能を持つインテリジェントシステムです：

  1. 会議議事録システム - 会議の音声認識と自動要約
  2. ディベートシステム - AIとのリアルタイム議論

📝 会議議事録システムの使い方

機能一覧

🎤 リアルタイム文字起こし

Web Speech APIを使用した音声認識で、会議内容をリアルタイムで文字化します。

🤖 AI議事録生成

GPT-4を活用して、文字起こしから自動で要約、重要ポイント、アクション項目を抽出します。

📊 ステータス管理

会議のステータス（導入 → 共有 → 議論 → まとめ）を管理し、進行状況を可視化します。

🏷️ スタンプ機能

文字起こしに重要・アクション・決定などのタグを付与して、内容を整理できます。

💬 コメント機能

文字起こしの各部分に対してコメントを追加し、詳細な説明や注記ができます。

🖍️ マーキング機能

文字起こしに色付けハイライトを追加し、重要な箇所を視覚的に整理できます。

⏱️ タイムキーパー機能

会議の時間を自動管理し、各フェーズの進行状況をリアルタイムで追跡します。

🎯 AIファシリテーター

会議の進行をサポートするAIアシスタント。議論を促進し、建設的な対話を支援します。

👥 複数AIメンバー

複数のAIキャラクター（最大8人）を会議に参加させ、多角的な視点からの意見を得られます。

🔊 テキスト読み上げ

VOICEVOXを使用したハイクオリティな音声合成で、議事録を自然な日本語で読み上げます。

操作手順

  1. 会議一覧ページから「新しい会議を作成」をクリック
  2. 会議タイトルを入力（例：「営業会議 2026/1/28」）
  3. 会議ルームに入室し、マイク許可を与える
  4. 会議ルーム内の機能設定：
    • AIファシリテーターの有効化：トグルボタンで「AIファシリテーター機能を有効にする」をオン。会議進行をサポートするAIが活動します
    • 複数AIメンバーの追加：右側のAIメンバーパネルから新規追加
      • 💡 新たなアイディアを提案：創造的な意見を提案するAI
      • 🎯 議論を促進：ファシリテーター型AI（通常1人のみ）
      • 😊 明るい：前向きで励ましのコメントをするAI
      • 😟 ネガティブ：批判的視点を提供するAI
      • 😠 怒りっぽい：強い異議を唱えるAI
    • 最大8人までのAIメンバーを同時に参加させることができます
    • 各AIメンバーは会議進行中にコメントや提案を自動生成します
  5. 「録音開始」ボタンをクリックして会議開始
  6. 会議進行中：
    • 音声が自動的に文字起こしされます
    • AIメンバーやAIファシリテーターの提案が表示されます
    • 重要な部分はスタンプで標記できます
    • 文字起こしを色付けマーキングで強調表示できます
    • 質問やコメントがあればコメント機能を使用
    • 各フェーズの進行状況を確認
  7. 「録音終了」をクリック
  8. AIが自動処理を開始（処理中状態）
  9. 処理完了後、議事録詳細ページで：
    • 完全な文字起こし
    • AI生成の要約
    • 重要ポイント一覧
    • アクション項目と担当者
    • 決定事項

🤖 AIメンバーとAIファシリテーター機能

AIメンバーについて

複数の異なるパーソナリティを持つAIを会議に参加させ、多角的な視点から議論をサポートします。

  • 💡 新たなアイディアを提案：創造的で革新的な提案をします
  • 🎯 議論を促進（ファシリテーター）：会議の進行をサポートし、建設的な対話を促進します
  • 😊 明るい：前向きで励ましのコメントを提供します
  • 😟 ネガティブ：批判的な視点からの指摘を行います
  • 😠 怒りっぽい：強い意見や異議を唱えます

特徴：

  • 最大8人のAIメンバーを同時に会議に参加させられます
  • 各AIメンバーの名前はカスタマイズ可能です
  • 会議中にいつでもAIメンバーを追加・削除できます
  • 各AIの発言がリアルタイムで表示されます
  • 議事録に全AIメンバーの発言が記録されます

AIファシリテーター機能

会議の進行をサポートするAIファシリテーター。議論を促進し、参加者全員の意見を引き出すのに役立ちます。

  • 会議進行のサポート：各フェーズの進行を管理し、時間配分を最適化します
  • 意見の整理：出された意見を整理し、重要なポイントを強調します
  • 質問の提示：参加者の思考を促す質問を提示します
  • 議論の活性化：沈黙を避け、建設的な議論を促進します
  • 有効化方法：会議ルームの設定パネルで「AIファシリテーター機能を有効にする」をオンにします

🔊 テキスト読み上げ機能（VOICEVOX）

読み上げ機能について

VOICEVOXを使用した高品質な日本語テキスト読み上げ機能です。議事録や要約を自然な日本語音声で聞くことができます。

  • 高品質な音声：VOICEVOXの複数のスピーカーから選択できます
  • 自然な発音：感情表現豊かで自然な日本語の読み上げ
  • 再生制御：再生速度の調整、一時停止、再開が可能
  • 複数スピーカー対応：異なるスピーカーを選択して、読み上げの印象を変えられます
  • 利用対象：議事録の要約、重要ポイント、アクション項目などが読み上げられます

読み上げ機能の使用方法

  1. 処理完了後の議事録詳細ページを開きます
  2. ページ上部の「🔊 読み上げ」ボタンをクリック
  3. スピーカーを選択（複数のVOICEVOXスピーカーから選択可能）
  4. 読み上げ対象を選択：
    • 要約全体
    • 重要ポイント
    • アクション項目
    • 決定事項
  5. 「再生」ボタンをクリックして読み上げを開始
  6. 再生中は以下の制御が可能：
    • 一時停止 / 再開
    • 再生速度の調整（0.5倍～2倍速）
    • 停止

🎯 ディベートシステムの使い方

機能一覧

🤖 複数のAIタイプ

論理的・創造的・外交的・攻撃的の4つのAIスタイルから選択できます。

⚔️ ターン制ディベート

ユーザーとAIが交互に意見を述べ、設定したターン数まで議論を続けます。

🏆 AI判定

ディベート終了後、AIが客観的に論点整理と勝敗判定を行い、詳細な評価を提示します。

📱 リアルタイム表示

ディベートの進行状況がリアルタイムで画面に表示されます。

操作手順

  1. ディベート一覧ページから「新しいディベートを作成」をクリック
  2. 以下の設定を入力：
    • テーマ：ディベートのテーマ（例：「テレワークは生産性を向上させるか」）
    • AIタイプ：対戦相手のAIスタイルを選択
      • 論理的：データと論理に基づいた議論
      • 創造的：ユニークな視点からの議論
      • 外交的：相手の意見を尊重した協調的な議論
      • 攻撃的：強い主張で相手を圧倒する議論
    • 最大ターン数：ディベートの長さを設定（デフォルト：3ターン）
  3. 「ディベート開始」をクリック
  4. ディベートルームで：
    • テーマとAIタイプを確認
    • 先攻と後攻の割当を確認（システムが自動決定）
  5. ディベート進行中：
    • 自分の番になったら、テキストボックスに意見を入力
    • 「送信」ボタンで自分の意見を提出
    • AIの意見が表示されるまで待機
    • AIの意見を読んで、反論を準備
    • ターン数に達するまでこれを繰り返す
  6. ディベート終了後：
    • 判定結果が表示（ユーザー勝ち・AI勝ち・引き分け）
    • 判定理由：AIが論点整理と勝敗の理由を詳細に説明
    • 全発言の記録を確認

⚙️ システム情報

技術スタック

  • バックエンド: Django 5.0
  • フロントエンド: HTML5 / CSS3 / JavaScript
  • リアルタイム通信: Django Channels / WebSocket
  • データベース: SQLite
  • AI: OpenAI GPT-4 / Web Speech API
  • キャッシング: Redis

必要な環境設定

  • モダンブラウザ（Chrome / Firefox / Safari / Edge）
  • マイク使用許可
  • OpenAI APIキー（.envファイルに設定）
  • インターネット接続

トラブルシューティング

  • 文字起こしが動作しない：ブラウザのマイク許可を確認してください
  • AI処理が遅い：ネット接続とAPIクォータを確認してください
  • ディベートが開始できない：ページをリロードして再度試してください
  • その他の問題：ブラウザのコンソール（F12）でエラーメッセージを確認
